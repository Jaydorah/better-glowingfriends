# Better Glowing Friends
Normal glowingfriends has 
/glowingfriends add PLAYER
this has both **enemy(red), ally(yellow), friend(green)** outlines
## usage
/glowingfriends ally add PLAYER

/glowingfriends enemy add PLAYER

/glowingfriends friend add PLAYER
